# RD63_Vespa

**Drone Designator**: RD-63

This repository serves as the source of truth for the RD-63 "Vespa" drone - a 3.5" HDZero cinewhoop build for light freestyle, cruising, and fun flying.

## Current Configuration

- **Frame**: SpeedyBee Bee35 (3.5" ducted cinewhoop, 153mm wheelbase)
- **FC/ESC**: HDZero Halo Stack (H743, 70A, integrated ELRS Gemini RX)
- **Motors**: EMAX ECO II 2004 3000KV (x4)
- **Props**: Gemfan Hurricane 3525 (tri-blade)
- **Video**: HDZero Race V3 VTX + Nano 90 V2 Camera
- **Antenna**: u.FL - TBD from existing stock
- **Receiver**: Integrated ELRS 2.4GHz Gemini (Halo Stack)
- **Batteries**: 4x Tattu 850mAh 4S HV + 4x Tattu 650mAh 4S (XT30)
